# zentinel

zentinel is a Zig-native mutation testing framework built through the sequential agent task system in this repository.

Start here when working as an agent:

- `AGENTS.md` for repository-level operating rules.
- `docs/AGENT_GUIDE.md` for task execution and handoff discipline.
- `tasks/STATUS.md` for the current active task and next handoff.
- `tasks/QUEUE.md` and `tasks/queue.json` for execution order.
